The dataset contains 5011 students, with each student's records spanning 6 consecutive semesters, and each semester recording grades for 3 courses.
student_performance.xlsx contains the raw data. input_target.npz includes two arrays: input.npy and target.npy, serving as the input data and prediction targets respectively. Their shapes are (5011, 4, 3) and (5011, 2, 3).
